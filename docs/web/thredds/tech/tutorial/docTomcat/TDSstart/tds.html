- Needs to be done as user tomcat 

- In this tutorial:
${tomcat_home} = /opt/tomcat
${tomcat_version} = apache-tomcat-5.5.17

- Go to webapps directory
% cd ${tomcat_home}/${tomcat_version}/webapps
ie /opt/tomcat/apache-tomcat-5.5.17/webapps

- download latest thredds.war

- manual intall:
	- stop tomcat
	% ${tomcat_home}/${tomcat_version}/bin/shutdown.sh
	- remove old TDS installation if there is one
	% cd ${tomcat_home}/${tomcat_version}/webapps
	% rm -r thredds
	- remove tomcat logs
	% rm ${tomcat_home}/${tomcat_version}/logs/*
	- start tomcat
	% ${tomcat_home}/${tomcat_version}/bin/startup.sh

- alternatively one can use the Tomcat Manager once the tomcat users have been 
  configured. Here's the default tomcat-users.xml:

<!--
  NOTE:  By default, no user is included in the "manager" role required
  to operate the "/manager" web application.  If you wish to use this app,
  you must define such a user - the username and password are arbitrary.
-->
<tomcat-users>
  <user name="tomcat" password="tomcat" roles="tomcat" />
  <user name="role1"  password="tomcat" roles="role1"  />
  <user name="both"   password="tomcat" roles="tomcat,role1" />
</tomcat-users>

Using a browser, go to:

http:/localhost:8080

click on management, tomcat passwd tomcat
