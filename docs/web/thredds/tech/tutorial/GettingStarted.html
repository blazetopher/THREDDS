Getting Started

Install Java

The TDS requires Java 1.5 or above, we recommend the current verison of 1.6 and use it on our development servers.

Current versions of the JRE and JDK are available from the Java web page, http://java.sun.com/.

The Java SE Developer Kit (JDK) 1.5.0_12 is already installed on the workshop computers at /usr/java/jdk1.5.0_12.

  1. Set JAVA_HOME environment variable
    • set JAVA_HOME /usr/java/jdk1.5.0_12

Install Tomcat

Current versions of Tomcat are available from the Tomcat web page, http://tomcat.apache.org/.

Install Tomcat 6.0:

  1. Move to installation location
    • cd tds
  2. Unpack the Tomcat 6.0.13 archive
    • tar xvzf /opt/apache-tomcat-6.0.13.tar.gz
  3. Set TOMCAT_HOME environment variable
    • set TOMCAT_HOME /home/workshop/tds/apache-tomcat-6.0.13
  4. Move to Tomcat bin directory
    • cd ${TOMCAT_HOME}/bin
  5. Edit startup.sh and shutdown.sh and set environment variables:
    • Run editor of choice, e.g.
      • vi startup.sh shutdown.sh
      • gedit startup.sh shutdown.sh
    • JAVA_HOME="/usr/java/jdk1.5.0_12"
      export JAVA_HOME
      J      AVA_OPTS="-Xmx1024m -Xms512m -server"
      export JAVA_OPTS
      C      ATALINA_HOME="/home/workshop/tds/apache-tomcat-6.0.13"
      export CATALINA_HOME

Running Tomcat

  1. cd to Tomcat bin directory
    • cd ${TOMCAT_HOME}/bin
  2. Startup Tomcat
    • ./startup.sh
  3. Open browser window and check that Tomcat is running
    • URL: http://localhost:8080/
    • You should see the Tomcat start page which begins with the text:

      If you're seeing this page via a web browser, it means you've setup Tomcat successfully. Congratulations!


  4. Shutdown Tomcat
    • ./shutdown.sh

Deploy TDS (thredds.war)

The THREDDS Data Server (TDS) is available from the Unidata FTP site at ftp://ftp.unidata.ucar.edu/pub/thredds/3.16/.

  1. Download the thredds.war file
    • Click on the above FTP URL.
    • Click on the thredds.war file and save to the Tomcat webapps directory, ${TOMCAT_HOME}/webapps.
  2. Startup Tomcat
    • ./startup.sh
  3. Test TDS in browser
    • URL: http://localhost:8080/thredds/
    • Browse distribution test catalogs and datasets
