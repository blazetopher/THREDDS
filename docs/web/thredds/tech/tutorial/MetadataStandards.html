Metadata Standards

There are a number of existing metadata standards available for describing datasets. These include:

  • Dublin Core - general library discovery metadata standard
  • FGDC/DIF - standard for geophisical data
  • ISO 19??? - standard for geophysical data (FGDC is merging/synchronizing with this ISO standard)
  • ADN - Digital library standard with education specific fields.

Including Existing Metadata Records in THREDDS catalogs.

Any metadata records can be included in or referenced from a THREDDS catalog from a metadata element.

Metadata can be included directly in a metadata element. The only requirement is that the XML be in a non-THREDDS namespace. For example:

<metadata>
</metadata>

Referencing a metadata record is done with XLink attributes:

<metadata xlink:title="" xlink:href="" />
