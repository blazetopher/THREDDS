TDS Training Workshop Preparations

Ghost Machine Prep

  • Add link on desktop quick start bar for Firefox and Terminal.
  • Add to Firefox:
    • Install "Live Http Headers" Add-on
    • Add "TDS Docs" folder for bookmarks and add:
      • "TDS Home Page": http://www.unidata.ucar.edu/projects/THREDDS/tech/TDS.html
      • "TDS Administrator Tutorial": http://www.unidata.ucar.edu/projects/THREDDS/tech/tds4.2/tutorial/schedule.html
      • "TDS Reference": http://www.unidata.ucar.edu/projects/THREDDS/tech/tds4.2/reference/
    • Open the the "TDS Docs" bookmarks in separate tabs and set them as the Firefox "Home Pages"
    • Add "TDS Instances" folder for bookmarks and add:
      • "TDS localhost:8080": http://localhost:8080/thredds/catalog.html
      • "TDS motherlode:8080": http://motherlode.ucar.edu:8080/thredds/catalog.html
      • "TDS motherlode:9080": http://motherlode.ucar.edu:9080/thredds/catalog.html
    • Webstart ToolsUI
      • Add "ToolsUI webstart" bookmark
      • Launch so that it is downloaded and cached.
      • Open the motherlode:8080 catalog in the THREDDS Catalog Chooser tab.
    • Webstart IDV
      • Add "IDV webstart" bookmark
      • Launch so that it is downloaded and cached.
      • Open the motherlode:8080 catalog in the Dashboard - "Data Choosers" tab - "Catalogs" page.
