Static Catalogs

These are the catalog roots and their referenced catalogs. They must live under ${tomcat_home}/content/thredds. What about TDR ??

Catalog Caching

Static catalogs are cached in memory. The expires attribute is used to decide whether to reread. If no expires attrinute, the catalog is never reread.
