Location of Various TDS Directories

You may wish to move some of the heavily used TDS directories to larger and more high performance disks. For instance:

  • Data directories
  • The content/thredds directory.
  • The Tomcat logs directory.
  • The TDS logs directory.

The easiest way to do this is with symbolic links. For instance, if Tomcat is installed on a smaller and/or lower performance disk (say /opt) but /data is a larger and higher performance disk, your ${TOMCAT_HOME} directory might look something like this:

my.server.edu: cd /opt/tdsA
my.server.edu: pwd
/opt/apache-tomcat-6.0.29-tdsA
my.server.edu: ll

And your data paths may all point into /data/pub.
