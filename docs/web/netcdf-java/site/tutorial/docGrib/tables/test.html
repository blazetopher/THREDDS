1.3 Testing your new GRIB tables/parameters

The easiest way to test your additions would be to dump your file using the Grib(1|2) routine, verifying your new tables/parameters. If you used Method 1 there are lines that are commented out in the GribDump code that need to be uncommented that adds the new tables/parameters. Method 2 and 3 don't require are code changed to Grib1Dump or Grib2Dump code. 


java -Xmx256m ucar/grib/grib1/Grib1Dump "your test file" | less

or

java -Xmx256m ucar/grib/grib2/Grib2Dump "your test file" | less
