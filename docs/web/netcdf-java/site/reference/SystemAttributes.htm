System Attributes

John Caron

last updated April 25, 2005

_unsigned

Indicates a Variable is unsigned, used only for byte, short, and int. Only valid value is true.

 

_dataType

Indicates the "scientific data type" of the dataset, currently understood values are Grid, Image, Point, Radial, Station, and Trajectory.
