NcML-2.2 Reference

Variables that get aggregated

  • Union: none, all variables from all datests are simply logically placed in the result dataset
  • joinExisting: all variables that have the aggregation dimension as the outer dimension
  • joinNew: only variables explicitly specified in a variableAgg element
  • Tiled:
  • FMRC: any variable that has a grid coordinate system, ie is recognized as a Grid by the CDM.
  • FMRCsingle: any variable that has a grid coordinate system, ie is recognized as a Grid by the CDM.

Reading Files to get Initial State

  • Union: must read each file
  • joinExisting: read one file as prototype, read time coordinate variable of each file.
  • joinNew, joinExistingOne: read one file as prototype, assign time coordinate variable for each file with a filename matcher
  • Tiled: read one file as prototype, assign sections for each file manually
