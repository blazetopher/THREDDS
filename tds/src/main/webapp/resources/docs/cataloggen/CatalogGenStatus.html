THREDDS Catalog Generator - Status

Comments to Ethan Davis or THREDDS mail list

The THREDDS Catalog Generator can produce a THREDDS catalog by scanning or crawling one or more local or remote dataset collections. The information it needs to know about the dataset collection and the resulting catalog are provided by a configuration document (see the CatalogGenConfig specification). The Catalog Generator can be run as a Java Servlet or as a stand-alone Java application. The application reads a single configuration document and produce a THREDDS catalog. The servlet provides 1) scheduling of all the catalog generation tasks being performed by the servlet, and 2) a web interface to control the configuration of the CatalogGen servlet.

Current Stable Release

THREDDS Catalog Generator version 0.8.1 (released 14 June 2004).

This software can produce THREDDS version 0.6 or 1.0 catalogs. The configuration files are described in the CatalogGenConfig specification document.

To download, install, and run the Catalog Generator, go to the installation page.
